# EKS Cluster Setup with Terraform

This setup creates an Amazon EKS cluster using Terraform. The configuration consists of three files:

- `main.tf`: Contains the EKS cluster and networking setup.
- `variables.tf`: Defines customizable variables.
- `outputs.tf`: Displays key details post-deployment.

## Commands

1. **Initialize**:  
   ```bash
   terraform init

2. **Plan**:
   ```bash
   terraform plan

3. **Apply**:
   ```bash
   terraform apply

4. **Destroy**:
   ```bash
   terraform destroy

Run these commands in sequence to provision and manage your EKS cluster on AWS.
